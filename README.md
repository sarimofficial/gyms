# Florida Insurance Estimator — React + Vite

This project is built with **React (Vite)** for fast development and optimized performance.
It powers the **Florida Insurance Estimator** website, featuring smooth animations, responsive layouts, and modern component architecture using **TailwindCSS** and **Framer Motion**.
![Website Screenshot](./public/images/display.png)

---

## 🚀 Tech Stack

* **React 18** — Component-based UI
* **Vite** — Lightning-fast build tool
* **TailwindCSS** — Utility-first responsive styling
* **Framer Motion** — Modern animation library for React
* **Lucide Icons** — Lightweight SVG icons
* **ESLint** — Linting and code consistency
* **DM Sans** — Primary typography

## 🧑‍💻 Development Setup

### 1. Clone the repository

```bash
git clone https://github.com/Appvertices/FIE-Website.git
cd FIE-Website
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Vite will start a local dev server (default at **[http://localhost:5173](http://localhost:5173)**).

---

## 🗷️ Build for Production

```bash
npm run build
```

The optimized static output will be generated inside the `dist/` folder.

---

## 🧩 Linting

```bash
npm run lint
```

Linting ensures code consistency and follows best practices for React projects.
